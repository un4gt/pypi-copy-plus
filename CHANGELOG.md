# 更新日志 / Changelog

## v1.1.0 - 2024-11-18

### 新功能
- 🎨 添加主题系统（浅色/深色/跟随系统）
  - 右上角图标按钮快速切换
  - 图标随主题变化（☀️ 浅色 / 🌙 深色 / 🔄 跟随系统）
  - 主题设置自动保存
- ⚡ 设置更改后无需刷新页面即可生效
- 💎 全新的现代化 UI 设计
  - 简洁的单栏布局
  - 专注于核心功能

### 优化
- 移除冗余的 `installCommand`
- 改进命令匹配逻辑，支持所有包管理工具
- 添加智能缓存机制
- 优化交互体验和视觉反馈
- 主题切换更加便捷和直观
